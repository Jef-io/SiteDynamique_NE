Développement

LAMP

Diminutif correspondant à "Linux / Apache / MySQL / PHP" C’est une solution d’hébergement de serveur ayant pour seul prix un ordinateur. Le stack LAMP est très répandu, permet de gérer un site ainsi qu’une base de données.

MERN / MEVN / MEAN

Ces acronymes représentent une pile de logiciels JavaScript gratuits et open-source pour la création de sites Web dynamiques et d'applications Web.

  • MongoDB
  • Express.Js
  • React
  • Vue
  • Angular
  • Node.js

GIT

C'est super git ! Git permet de coder de manière collaborative.

  • GitHub
  • GitLab
  • BitBucket
  • GitKraken

Intégration continue

L'intégration continue est un ensemble de pratiques utilisées en génie logiciel consistant à vérifier à chaque modification de code source que le résultat des modifications ne produit pas de régression dans l'application développée. - Wikipedia

  • Jenkins
  • Circle CI
